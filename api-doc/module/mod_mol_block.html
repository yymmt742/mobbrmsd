type, public, sequence  ::  mol_block

molecular block indicator atomic coordinates vector must be stored in the following format. X(d,m,n) - d :: spatial dimension. - m :: number of atom in a molecule. - n :: number of molecule. - where X(d,:f,:g) :: Free rotatable. - X(d,f+1:,g+1:) :: Fixed.
