Factorial tree for branch and bound.
This structure handles factorial trees with nodes of constant memory size, ld.

The factorial tree is completely defined by the number of level and a constant . A node in level has childs, and each level has nodes. Here, only nodes for level are kept as state vector. All other information is discarded.

The tree informations are kept by header q(*) and state vector s(*).

Note

q(*) constant and should be treated as an immutable variable.
However, it is defined as an integer array so that it can be treated as part of a data structure at a higher level imprementation.

The tree type has states and , where is the current level and is the selected node indices. Let be current node, and a retained nodes belonging to is defined as a -queue.

A -queue has one of the following states, corresponding to the state of the current node.
- Node is selected ::
- -queue is unexplored ::
- -queue is explored ::

The real data is kept in an external heap in the form W(ld, *). The value stored in W(1, *) is treated as a node evaluation value. (to be implemented by the user).

Note

W can be updated dynamically, but changes to the memory referenced by nodes in the hierarchy below will destroy the tree structure. The user should control when to update W.
