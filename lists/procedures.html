Hungarianmod_HungarianSubroutine

Calculate the minimum linear assignment cost using Hungarian method. If m and n are different, the sum of the linear assignments of the smaller is returned. If m>0 and n>0, W(1) stores the minimum linear assignment cost. If m==0 or n==0, do nothing. If (m<0 or n<0) and (|m|>0 and |n|>0), W(1) stores the required memory size for C(|m|,|n|).
